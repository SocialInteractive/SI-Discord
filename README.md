## Getting Started

### Dependencies

* Newest Windows 10 Version (Version	10.0.19043 Build 19043)
* Newest Discord Version (Stable 89709)

### Installing

* Download [Discord](https://discord.com)
* Join our [Official Discord Server](https://discord.io/socialint)
* Done!
